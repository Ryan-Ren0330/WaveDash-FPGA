Visualization of population representations¶

Data used to construct the TSNE visualization of (overlapping) population representations of phonemes, syllables, and morphemes are given in Fig3-population_representations.xlsx. The GLM model AIC and D2 values are given in two different sheets; these values are used to organize the position of cells in TSNE space according to shared representational characteristics (model goodness-of-fit).

tsne = TSNE(random_state=0, 
            perplexity=30, 
            early_exaggeration=6, 
            n_iter=2000, 
            n_iter_without_progress=500, 
            metric='cosine')

t = tsne.fit_transform(d2)

theta = np.radians(90)
rotation_matrix = np.array([
    [np.cos(theta), -np.sin(theta)],
    [np.sin(theta), np.cos(theta)]])
_t = np.dot(t, rotation_matrix.T)

node_points = np.array([(x, y) for x in np.linspace(min(_t[:,0]), max(_t[:,0]), 16) for y in np.linspace(min(_t[:,1]), max(_t[:,1]), 17)])
C = cdist(_t, node_points, 'euclidean')
r, assignment = linear_sum_assignment(C)

fig, axes = plt.subplots(1,3,figsize=(10,3))

for x,ax in enumerate(axes.ravel()):

    # Size scales with d2
    _s = d2[:,x]*150
    
    # Color identifies the feature
    _c = np.zeros(c.shape)
    _c[:,x] = 1

    # Alpha scales with likelihood of model correctness
    norm = mpl.colors.Normalize(vmin=0, vmax=5, clip=True)
    _c[:,-1] = 1-norm(aicdiff[:,x]).data
    
    ax.axis('off')
    ax.scatter(node_points[assignment,0], node_points[assignment,1], c=_c, s=_s)
